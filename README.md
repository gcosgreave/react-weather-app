# React JS Weather Widget

A simple React app that uses openweathermap's API to get the weather of any location. :sunny: :umbrella: :cloud:

## Installation

```console
  cd weather-app
  yarn install
  yarn run
  
```

## Usage

Fill out the city and country then simply hit "Get Weather". :boom:
